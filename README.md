# 🛒 DigiTools — Digital Tools Marketplace

A modern digital tools buying platform built with React and Tailwind CSS. Browse, select, and purchase premium digital products to supercharge your workflow.

---

## 🚀 Live Site

🔗 [Live Demo](https://mellow-longma-a133af.netlify.app/)

---

## 🛠️ Technologies Used

| Technology | Purpose |
|---|---|
| React.js | UI Framework |
| Tailwind CSS v4 | Styling |
| DaisyUI v5 | UI Components |
| React Router DOM | Navigation & Active Links |
| React Toastify | Toast Notifications |
| React Icons | Icon Library |
| Vite | Build Tool |
| JSON | Product Data |

---

## ✨ Features

### 🛒 Smart Cart System
Add products to cart, remove individual items, and proceed to checkout — all with smooth toast notifications powered by React Toastify.

### 🔄 Dynamic Product & Cart Toggle
Switch between the Products view and Cart view instantly with a clean toggle button system. Cart count updates live in the Navbar.

### 📱 Fully Responsive Design
Optimized for all screen sizes — mobile, tablet, and desktop — with a collapsible hamburger menu on smaller devices.

## 📦 Installation & Setup

```bash
# Clone the repository
git clone https://github.com/golamrabbi73/DigiTools-Platform.git

# Go to project folder
cd DigiTools-Platform

# Install dependencies
npm install

# Start development server
npm run dev
```
